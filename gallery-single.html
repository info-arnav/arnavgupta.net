Blink

is the most basic programme that can be made . This is made up of an arduino and LED. we simply connect the LED to t=any one of the digital pin and other pin to the ground. after this we mply add our code to the arduino
below is the code that was uploaded to arduino for this task:

void setup() {
// initialize digital pin LED_BUILTIN as an output.
pinMode(13, OUTPUT); //13 is used because i attached the led to pin 13
}

// the loop function runs over and over again forever
void loop() {
digitalWrite(13, HIGH); // turn the LED on (HIGH is the voltage level)
delay(1000); // wait for a second
digitalWrite(LED_BUILTIN, LOW); // turn the LED off by making the voltage LOW
delay(1000); // wait for a second
}

Smart Bin

Smart bin is a project made up of integration of arduino and raspberry pi. the dustbin is such that it can detect people and hence open on its own. After the waste is thrown, with the help of moisture sensor the waste is segregated with respect to dry or wet waste.
later on the waste is converted into manure via deposition of soul and high pressure surrounding. After the bin is completely filled up a message is sent to the MCD with the dustbin location with a request to collect the bin, this is done with the help of twillio.
Further with the help of Raspberry pi servo, motion sensor and camera interfacing is done.
